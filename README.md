# box
推箱子

## 介绍
使用canvas和原生JavaScript实现的推箱子游戏，地图可以通过 js/maps.js 文件来修改。

拥有一个地图编辑模式，可以使用鼠标单击格子来修改格子类型。

自定义地图保存时，仅仅从控制台输出地图，并未对 js/maps.js 地图文件修改。
